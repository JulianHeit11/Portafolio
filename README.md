## ▶️ Cómo ejecutar el proyecto

1. Clonar el repositorio

```bash
git clone https://github.com/Fausto-Desch/mi-portfolio.git
```

2. Instalar dependencias

```bash
npm install
```

3. Ejecutar en modo desarrollo

```bash
npm run dev
```
